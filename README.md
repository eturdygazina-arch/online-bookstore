ZEIN Online Bookstore
Overview
ZEIN Online Bookstore is a full-stack web application developed as a university project using Flask, MongoDB, HTML, CSS, and JavaScript.
The platform provides a complete online bookstore experience, allowing users to browse books, manage profiles, place orders, save payment cards, receive notifications, purchase gift certificates, and register for educational courses.
The user interface is implemented in the Kazakh language.

⸻


Project Architecture
The application follows a client-server architecture:
User → Flask Backend → MongoDB Database → HTML Templates → User Interface
The backend handles business logic, authentication, order processing, notifications, and communication with MongoDB.
